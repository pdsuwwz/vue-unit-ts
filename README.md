Unit Testing for Vite 4.x + Vue 3.3 + TypeScript + Jest 29.x


## Environment Support

* Node >= 16.15.x

## Install

Install node dependencies in all packages

```bash
pnpm install
```

## Run

Local Development

```bash
pnpm dev
```

## Test

Unit Testing

```bash
pnpm test
```

Test code coverage

```bash
pnpm test:coverage
```
